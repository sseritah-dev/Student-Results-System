# Student Results Management System
A python application for managing student results.
It allows users to add students, calculate averages, assign grades and view class statistics.

FEATURES:
Add new students with marks for multiple subjects,
Automatically calculate average score,
Assign grades based on performance(A+, A, B, C, D, and F),
View all students in a structured format,
Search for individual students by name and by grade,
View student rankings (highest to lowest),
Edit and delete student records,
store and load data using JSON file.

CONCEPTS USED:
Variables and data types,
Functions,
Lists and dictionaries,
Loops(for/while),
Conditional statements(if/elif/else),
File handling(JSON) and user input handling,
Menu-driven programs.

HOW IT WORKS:
1. User runs the program
2. Main menu is displayed
3. User selects an option
4. Program performs an action (add, search, view, edit, delete etc)
5. Data is saved automatically in the JSON file.

TECNOLOGIES USED:
Python 3,
JSON (for data storage),
Command line interface (CLI).

FUTURE IMMPROVEMENTS:
